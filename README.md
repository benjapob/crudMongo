# crudMongo

Crud para libros hecho para el ramo de Bases de Datos no Estructuradas.

## Instalación

* Instala MongoDB Community Server
    
Ejecuta estos comandos en terminal

* `python app.py`

[Abrir la página](http://127.0.0.1:5000/)

## Tecnologías
* Python
* Flask
* Bootstrap
* JavaScript
* MongoDB
